### 快速启动
**保证你的机器上的node版本 >= 7.0 和 NPM >= 3**

```使用
# 克隆下来代码
# --depth 1 参数只拿到最新版本，不要历史提交纪录
git clone --depth 1 git@git.oschina.net:plothao/RM_BQ_EV200_006.git

# 改变 文件夹 到咱们的仓库
cd RM_BQ_EV200_006

# 用npm安装依赖
npm install

# 启动服务器
npm start


# npm如果不要用，就用cnpm
# https://github.com/cnpm/cnpm
```
在浏览器中打开 [http://localhost:8800](http://localhost:8800)
